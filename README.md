## :running: Would you like to run this project locally?

Open terminal and follow my instruction:


1) Clone repository:

```
https://github.com/front-end-by-rimantas/48-grupe-team-vinted
```

2) Install NPM packages:

```
npm install 
```

3) Run the server:

```
npm run dev
```
